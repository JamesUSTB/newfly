# 更新日志 / Changelog

## [1.0.0] - 2024-11-26

### 新增 (Added)
- ✨ 完整的数据预处理流程
  - 多文件拼接功能
  - 小波阈值去噪
  - 高通滤波和多项式去趋势
  - Savitzky-Golay平滑
  
- ✨ 事件检测与截取系统
  - 多传感器能量包络计算
  - 自适应阈值检测
  - 智能片段截取

- ✨ 全面的特征提取模块
  - 时域特征（14个/通道）
  - 频域特征（10个/通道）
  - 时频特征（17个/通道，小波分析）
  - MFCC特征（13个系数，适配振动信号）
  - 跨缝特征（8个/深度，传荷能力分析）

- ✨ 三种机器学习模型
  - LightGBM分类模型（速度等级预测）
  - LightGBM回归模型（连续速度预测）
  - PIRCN物理引导残差修正网络
  - 孤立森林健康状态检测器

- ✨ 可视化工具集
  - 时域波形图
  - 频谱图
  - 特征热图
  - 混淆矩阵
  - 特征重要性图
  - 异常分数分布图
  - 训练历史曲线

- ✨ 评估工具集
  - 分类评估（准确率、精确率、召回率、F1）
  - 回归评估（MAE、RMSE、R²、MAPE）
  - 模型对比功能
  - 交叉验证支持

- ✨ 辅助工具
  - 示例数据生成器
  - 安装测试脚本
  - 详细文档（README、快速开始指南、项目总览）

### 技术特性 (Features)
- 📊 支持多种实验工况（左板、右板、横跨）
- 🔄 支持多速度等级（40/60/80/100 km/h）

- 🎯 物理约束融合
